# Sistema de Gerenciamento - Padaria Pão e Confia

Sistema web para gerenciamento de produtos, categorias e cadastros básicos de uma padaria.

## Pré-requisitos

- PHP 7.4 ou superior
- MySQL 5.7 ou superior
- Servidor web (Apache/Nginx) ou PHP built-in server
